# fyllo motion

`fyllo motion` is a React + Vite creative video tool for building motion videos with prompt-driven generation, uploaded assets, AI-generated images, and AI-generated music.

## Features

- Prompt-driven video generation with Remotion
- Image and audio asset library
- AI image generation via Pixazo
- AI music generation via Pixazo Tracks
- Installable PWA support

## Run locally

Prerequisite: Node.js

1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in `.env.local`
3. Start the dev server:
   `npm run dev`

## Build

`npm run build`
